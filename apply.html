応募フォーム：設問テンプレ

以下はGoogleフォームなどでそのまま使える設問の例です。各質問はプロジェクトに合わせて自由に編集してください。

セクション1｜基本情報

  • 氏名（必須）: 短文。2文字以上の検証。
  • ふりがな（必須）: 短文。
  • メールアドレス（必須）: メール形式を検証。LINE招待リンクの送付に使用します。
  • 電話番号（必須）: 半角数字と「-」「+」可（例: 090-1234-5678）。
  • Instagram（任意）: @からでもURLでも可。
  • LINE ID（任意）: 後ほど公式アカウントの友だち追加リンクも送ります。不明な場合は空欄でOK。
  • 緊急連絡先（必須）: 段落回答。氏名／続柄／電話／メールを入力してもらいます。

セクション2｜出資費用の想定

  • 出資費用の想定（必須）: 選択式。1万円未満、1〜3万円、3〜5万円、5〜10万円、10万円以上。現時点の目安でOK。
  • 金額に関する補足（任意）: 段落回答。材料費に充当希望、分割相談可 など。

セクション3｜特技・スキル

  • 該当するスキル（複数選択可）: DIY、音楽、動画編集、写真撮影／ドローン、料理、デザイン、広報、企画運営、その他。
  • スキルの具体例・実績（任意）: 段落回答。例: インパクト・丸ノコ使用経験あり／動画編集で短編3本制作 など。

セクション4｜参加動機・自己紹介

  • 参加動機（必須）: 段落回答。最低100文字などの検証を設けると◎。
  • 自己紹介（必須）: 段落回答。得意なこと／性格／今回挑戦したい役割など。

セクション5｜同意事項

  • 写真・動画の撮影・公開への同意: 必須のチェックボックス。広報目的でSNS・Webに掲載することがある旨を伝えます。
  • 事故・怪我等に関する免責への同意: 必須のチェックボックス。主催者に重大な過失がある場合を除き責任を問わないことを了承します。
  • 個人情報の取り扱い: 必須のチェックボックス。入力情報はMakeCamp運営・連絡のみに使用し、目的外利用をしないことを明記します。

回答後にLINE情報を自動で送る方法

方法A（最も簡単）：確認メッセージにリンクを載せる

  1. Googleフォーム右上の設定（歯車）→ プレゼンテーション → 確認メッセージ を編集します。
  2. 以下のテキストを確認メッセージに貼り付けます。
ご応募ありがとうございます！🎉
次の手順で連絡用LINEに参加してください
1) 友だち追加リンク： https://lin.ee/XXXXX
2) 参加合言葉： 「MakeCamp応募済」
※LINE参加でエントリー完了となります

必要であれば、QR画像をフォーム末尾に挿入しておくと親切です。

方法B（おすすめ）：自動返信メールでLINEリンクを送る

フォームの回答に紐づいたスプレッドシートを利用し、Apps Scriptで自動返信メールを送る方法です。

  1. 回答スプレッドシートを開き、拡張機能 > Apps Script にアクセスします。
  2. 以下のコードを貼り付け、LINE_INVITE_URL を自身の招待リンクに変更します。
function onFormSubmit(e) {
  const resp = e.namedValues;
  const email = (resp['メールアドレス'] || resp['Email'] || [''])[0];
  const name  = (resp['氏名'] || [''])[0];
  if (!email) return;
  const LINE_INVITE_URL = 'https://lin.ee/XXXXX';
  const QR_IMAGE_URL    = 'https://drive.google.com/uc?export=view&id=XXXXXXXX';
  const subject = `【MakeCamp】応募ありがとうございます（LINEご参加のお願い）`;
  const body = `${name} 様\n\nMakeCampへのご応募ありがとうございます！\n\n連絡はLINEオープンチャット／公式アカウントで行います。\n下記より参加（友だち追加）をお願いします。\n\n▼LINE参加リンク\n${LINE_INVITE_URL}\n\n▼参加合言葉\nMakeCamp応募済\n\n${QR_IMAGE_URL ? '▼QRコード（画像URL）\n' + QR_IMAGE_URL + '\n' : ''}\n\n— MakeCamp運営\n`;
  MailApp.sendEmail({ to: email, subject, body });
}
  1. Apps Scriptのトリガー設定で onFormSubmit をフォーム送信時に実行するよう設定し、初回実行で権限を付与します。

この方法により、送信者のメールにLINE招待が自動送信されるため、確実に連絡導線を提供できます。

仕上げのチェックリスト

  • フォーム設定でメールアドレス収集をONにするか、設問で必須に設定します。
  • 確認メッセージにLINEリンクを掲載（方法A）。
  • 自動返信メール（方法B）も追加するとリマインド効果が高まります。
  • スプレッドシートの列の順番・見出しがスクリプトと一致しているか確認します。
  • QR画像をGoogleドライブに保存し、共有リンクを「リンクを知っている全員」に設定します。
  • 同意文言はプロジェクトのポリシーに合わせて必要なら追記してください。
